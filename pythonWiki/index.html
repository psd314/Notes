Operators

Additon:   +

>>> 3 + 5

8

Subtraction:   -

>>> 10 - 3

7

Multiplication:   *

>>> 12 * 8

96

Division:   /

>>>35.0 / 7.0

5.0

Exponentiation:   **

>>> 5 ** 5

3125

Floor Division:   //   (integer division without remainder)

>>> 7 // 3

2

Modulus:   %   (returns remainder)

>>> 7 % 3

1

Declaring a Variable

Variable names can be comprised of letters, numbers and underscores ( _ ). To declare a variable:

var_name1 = value


>>> x = 3

>>> x

3
